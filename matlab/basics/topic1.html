Gauss Elimination

Step-by-Step Explanation

  1. Step 1: Input the system of equations

    In this step, you need to input the coefficients of the variables and the constants in the system of equations. Let's assume you have a system of three equations with three variables as an example:

    A = [2, -1, 3; 1, 3, 1; 4, -6, 2];
B = [9; 10; 12];

  2. Step 2: Augment the matrices

    To simplify the Gaussian elimination process, we augment the matrix of coefficients with the matrix of constants. This creates an augmented matrix, which is a combined matrix of A and B:

    AB = [A, B];

  3. Step 3: Perform forward elimination

    In this step, we perform row operations to transform the augmented matrix into an upper triangular matrix (zeros below the main diagonal). We start by eliminating the first variable from the second and third equations. Then, we eliminate the second variable from the third equation.

    [n, m] = size(AB);
for k = 1:n-1
    for i = k+1:n
        factor = AB(i,k) / AB(k,k);
        AB(i,k:m) = AB(i,k:m) - factor * AB(k,k:m);
    end
end

  4. Step 4: Perform back substitution

    After performing forward elimination, the augmented matrix will be in the upper triangular form. Now, we can perform back substitution to find the values of the variables.

    X = zeros(n, 1);
X(n) = AB(n,m) / AB(n,n);
for i = n-1:-1:1
    X(i) = (AB(i,m) - AB(i,i+1:n)*X(i+1:n)) / AB(i,i);
end

  5. Step 5: Output the results

    Finally, we display the values of the variables.

    disp('Solution:');
disp(X);
